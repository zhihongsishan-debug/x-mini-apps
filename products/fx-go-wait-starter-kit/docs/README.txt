 クイックスタート
----------------------------------------------------------------

 1. ZIPを任意のフォルダに展開する
 2. paid-html/index.html をブラウザで開く
       -> 7条件をYES/NOで切り替えるだけで判定が出る
 3. MT5チャートで使う場合:
       a. mt5/FX_GO_WAIT_Checker.mq5 を MT5 の
          MQL5/Indicators/ にコピー
       b. MetaEditor で F7 でコンパイル
       c. ナビゲーター > インジケータ > カスタム から
          チャートにドラッグ
 4. 詳しい手順とトラブルシュートは docs/setup-guide.html を参照

----------------------------------------------------------------
 判定ロジック (要点)
----------------------------------------------------------------

 各条件 YES = +15点 / 最大 100点 (7点満点でも100にキャップ)

   80-100  -> GO       (RISK: LOW or MEDIUM)
   60- 79  -> CAUTION  (RISK: MEDIUM)
    0- 59  -> WAIT     (RISK: HIGH)

 詳細は docs/decision-rules.html を参照してください。
