2. 有料HTMLの開き方

  1. ZIPを任意のフォルダに解凍します。
  2. paid-html/index.html を ダブルクリック でブラウザに開きます（推奨ブラウザ: Chrome / Edge / Safari の最新版）。
  3. 7つの条件をタップ／クリックで YES に切り替えると、SCORE・GO/CAUTION/WAIT・RISK が即時更新されます。
  4. RESETボタンで全条件をクリアできます。
オフライン動作。外部APIや外部ライブラリは一切使用していないため、インターネット未接続でも問題なく動作します。

3. MT5インジケータの導入

3-1. ファイル配置

  1. MT5を起動し、メニューから ファイル → データフォルダを開く を選択。
  2. 開いたフォルダから MQL5/Indicators/ へ移動。
  3. 同梱の FX_GO_WAIT_Checker.mq5 を、その Indicators フォルダにコピーします。

3-2. コンパイル（MetaEditor）

  1. MT5上で F4キー を押す（または ツール → MetaQuotes Language Editor）。
  2. 左ペインの Indicators 配下から FX_GO_WAIT_Checker.mq5 を開きます。
  3. F7キー（または上部の Compile ボタン）でコンパイル。
  4. 下のログに 0 errors, 0 warnings と出れば完了。
コンパイル後、同名の .ex5 ファイルが Indicators 配下に自動生成されます。

3-3. チャートに適用

  1. MT5の ナビゲーター → インジケータ → カスタム を開く（表示されない場合はMT5を再起動）。
  2. FX_GO_WAIT_Checker をチャートにドラッグ＆ドロップ。
  3. パラメータダイアログで、現在のセットアップに合わせて Cond1_TrendMatch 〜 Cond7_TimeOK を true / false で設定。
  4. OKを押すと、チャート左上にダーク調のチェックリストパネル + SCORE + DECISION + RISK が表示されます。
  5. 条件を変更したいときは、チャート上のインジケータを右クリック → プロパティ から再編集できます。

3-4. パネル位置・色のカスタマイズ

入力パラメータ PanelX / PanelY でパネル位置、 ColorBG / ColorBorder / ColorOK など各色を変更できます。

4. よくあるエラーと対処

「コンパイルエラーが出る」

  • MT5が build 3815以降 であることを確認してください（MT4ではビルドできません）。
  • ファイル名は FX_GO_WAIT_Checker.mq5 のまま変更しないでください。
  • 配置先は必ず MQL5/Indicators/ です（Experts/ や Scripts/ ではコンパイルできてもインジケータとして表示されません）。

「ナビゲーターに表示されない」

  • MT5を一度再起動するか、ナビゲーターで右クリック → 更新。
  • ナビゲーターの カスタム 配下を確認してください。

「パネルが表示されない / 隠れている」

  • チャート左上が他のインジケータで埋まっている場合、PanelX を増やして右に、PanelY を増やして下にずらしてください（推奨：PanelX=400, PanelY=20 等）。
  • 同名のオブジェクトが残っている場合は、チャート右クリック → オブジェクト → 全オブジェクト削除 で初期化。

「条件をtrueにしても変わらない」

  • パラメータ変更後、OKを押して反映する必要があります（数値ボックスを編集しただけでは反映されません）。
  • 反映されない場合はインジケータを一度削除してからもう一度ドラッグしてください。
