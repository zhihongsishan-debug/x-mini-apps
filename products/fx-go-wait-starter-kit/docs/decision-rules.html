7条件 × スコア × GO / CAUTION / WAIT の意味と運用

1. スコアと判定の対応

YESにした条件 1つにつき +15点。最大表示スコアは 100（7点満点でも105ではなく100にキャップされます）。

2. 7条件の意味と YES / NO 判断基準

01上位足トレンドが一致している
エントリーする時間足の1〜2段階上の足のトレンド方向と、これからのエントリー方向が一致していること。例えば 5分足で買いたいなら、15分足・1時間足のトレンドが上昇方向であることを確認する。
YES: 上位足の高値・安値が切り上がり/切り下がりで明確 ｜ NO: レンジ／逆方向／判断不能
02押し目 / 戻り目が成立している
トレンド方向に沿って一度引いてからの再加速ポイントでエントリーできていること。直前の強い動きの真ん中を追いかけていないか確認する。
YES: 直近の押し / 戻りからの反応が確認できる ｜ NO: 直前から既に大きく走っている／反応待ち
03ブレイク確認がある
直近のレジスタンス／サポートを実体で抜けたこと、または明確な構造ブレイクがあること。「もう抜けるかも」「抜けそう」では NO。
YES: ロウソクの実体終値で抜けた／構造の高値安値更新 ｜ NO: ヒゲのみ／継続性なし
04ボラティリティが十分
直近のATRやレンジ幅が普段比で薄すぎないこと。指標前のスクイーズ、深夜の薄商いなどで足が動かない局面は条件不足とみなす。
YES: 直近20本のATRが平常水準以上 ｜ NO: 不自然に縮小／無風
05RR 1:1.5以上が見込める
想定する損切り幅に対して、利益確定までの距離が1.5倍以上取れること。直近高値・安値や次のサポレジまでの距離で算出する。
YES: SL 20pips ／ TP 30pips 以上が確保できる ｜ NO: 直近の壁が近すぎる／RRが1:1未満
06損切り位置が明確
「ここまで来たら間違いだった」と明確に説明できる損切りポイントがあること。直近スイング、トレンドラインの裏側など。なんとなくの pips 数では NO。
YES: 構造的に説明できる損切り位置がある ｜ NO: 雰囲気で決めている／毎回違う
07時間帯が悪くない
流動性が枯れる時間帯、重要指標発表直前、週末持ち越しに近い時間帯ではないこと。東京/ロンドン/NY のいずれかのアクティブ帯であることが理想。
YES: アクティブ時間帯／指標まで余裕あり ｜ NO: 重要指標前後数十分／流動性枯渇時間／週末直前

3. WAITの価値

勝率を底上げする最大の方法は「条件が揃わない場面でエントリーしない」ことです。 WAITは敗北ではなく、来るべきチャンスのために弾を残す判断です。

  • 無駄なエントリーを1日3回減らせば、月間で60回前後の負け要因が消えます。
  • 条件待ちは精神的に苦しくても、口座残高にとっては最も寄与するアクションです。
  • 本キットは「エントリーを取りに行く」ためのツールではなく、「エントリーしない理由」を明文化するためのツールです。

4. 使用例

CASE A — 5分足トレンドフォロー

5分足で米ドル円の押し目買いを検討。15分・1時間足ともに上昇トレンドで 条件1: YES、 直近の押し目で陽線反発で 条件2: YES、レジスタンスを実体で抜けて 条件3: YES、 ATR平常 条件4: YES、SL 12pips / TP 20pips で 条件5: YES、 直近スイングロー下に SL 設定で 条件6: YES、ロンドン序盤で 条件7: YES。 → GO / RISK: LOW / SCORE 100。ロット管理を守って執行。

CASE B — 中途半端な戻り売り

上位足が下降トレンド 条件1: YES、ただし 1分足の戻りで条件2は薄い。ブレイク無しで 条件3: NO、 ATR薄め 条件4: NO、SL 8pips / TP 10pips で 条件5: NO、SLは雰囲気 条件6: NO、 指標30分前 条件7: NO。 SCORE 30。 → WAIT。ここで入ると統計上の負けトレード。見送りが正解。
