DOOM STACK PRO — Quick Start
============================

60 seconds. Open the app. Do these in order.

1. WELCOME
   On first run a 3-step welcome appears.
   - Step 1: how to add a task.
   - Step 2: how the weight meter works.
   - Step 3: where to back up.
   Tap "Skip" or walk through. Either is fine — the app is self-
   explanatory.

2. ADD YOUR FIRST TASK
   - Type in the bottom input.
   - Pick a priority pill: HIGH / MID / LOW.
   - Tap "Add" (or press Enter).
   The task lands on the stack and the Mental Load percentage rises.

3. WATCH THE STACK
   The vertical tower above the filters shows your incomplete tasks
   as physical mass. HIGH blocks are thicker and sit at the bottom.
   The badge at bottom-right shows count and total weight (w).

4. COMPLETE A TASK
   Tap the round checkbox on the left of any task. The task fades and
   gets a strike-through. The undo toast at the bottom lets you
   reverse the action within 6 seconds.

5. EDIT DETAILS
   Tap the ✎ pencil on a task to open the edit sheet.
   - Title, priority, category, due date, repeat, notes — all here.
   - Tap "Save" to write changes.
   - Tap "Cancel" or the backdrop to dismiss.

6. NOTES + SUBTASKS
   Tap the ▾ chevron on a task to expand.
   - Notes display below the meta row.
   - Add subtasks from the inline input. Tap to toggle complete.

7. FILTER YOUR VIEW
   Top chip row: Today / Tomorrow / This week / Next week / All / Done.
   Tap the date pill on the right to filter by a specific date.

8. CHECK YOUR LOAD
   Top-right shows: percentage + tier (LIGHT / BUSY / HEAVY / DOOM).
   Below the meter: 4 cells — Today, Week, Month, Streak.

9. OPEN DOOM ANALYSIS
   Tap "Doom analysis ▾" between the stats strip and the stack tower.
   Inside:
   - Pressure (weighted unfinished burden, 0–100)
   - Procrastination flags (3+d, 7+d, High idle)
   - Time tendency (morning / day / evening / night / balanced)
   - Top categories
   - 14-day mental-load sparkline
   - This-week summary
   - Export JSON / Import JSON buttons

10. BACK UP YOUR WORK
    Open Doom analysis → Export JSON. A file named
    doom-stack-YYYY-MM-DD.json downloads. Store it anywhere safe.
    To restore: Doom analysis → Import JSON → pick the file.

THAT'S IT.
- 追加 → 整える → 完了 → 重さが軽くなる
- Add → arrange → complete → the stack lightens.
