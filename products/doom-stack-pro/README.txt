DOOM STACK PRO
==============

A premium minimal task & mental-load tracker.
"Your tasks have weight."

Single-file, offline, no account, no tracking.

WHAT'S INCLUDED
---------------
- index.html        : the application (open it in any modern browser)
- README.txt        : this file
- quick-start.txt   : 60-second first-run guide
- feature-guide.txt : full reference for every feature
- backup-guide.txt  : how export / import / data ownership work
- license.txt       : license terms (personal use)
- disclaimer.txt    : warranty, liability, and refund language

WHO IT'S FOR
------------
People who want a calm, premium daily ToDo with a real mental-load
view. Operators, founders, designers, knowledge workers — anyone whose
"head full of things" is the actual problem they're solving.

WHAT MAKES IT DIFFERENT
-----------------------
1. Weight, not just lists. Every task has a priority weight that adds
   to a Mental Load score (0–100%). Heavier work sits lower in the
   visual stack — you can see your load.

2. Doom analysis. Pressure score, procrastination flags (3+d / 7+d /
   high-priority idle), time tendency (morning vs. night), category
   tendency, and a 14-day mental-load sparkline. Built-in, local-only.

3. You own your data. Export / import the entire workspace as a JSON
   file. No cloud, no API, no account. Browser localStorage holds
   everything until you back it up.

4. Premium minimal UI. Black / graphite / soft gray palette, calm
   typography, single restrained accent. No neon, no arcade — built
   for daily use without visual fatigue.

QUICK START
-----------
Open index.html in a modern browser (Chrome, Safari, Firefox, Edge).
The app runs locally. The first run shows a 3-step welcome — skip or
walk through. Add your first task at the bottom, pick a priority, tap
Add. Tap a card's pencil to edit (notes, due date, repeat, subtasks).

See quick-start.txt for the 60-second tour.

PERSISTENCE
-----------
- Tasks + UI state: localStorage key 'doomStackPro:v1'
- Mental-load history: localStorage key 'doomStackPro:history:v1'
- Onboarding flag:    localStorage key 'doomStackPro:onboarded:v1'

Clearing site data wipes the workspace. Use Export JSON regularly.

REQUIREMENTS
------------
- A modern browser with localStorage and JavaScript enabled.
- No internet connection required after the first load.
- Recommended: a current desktop or mobile browser within the last 2
  major versions.
